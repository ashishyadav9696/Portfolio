# Ashish Kumar Yadav — Professional Portfolio

A state-of-the-art personal portfolio website built with **React** and **Vite**, designed to showcase top-tier full-stack engineering expertise in the **MERN** ecosystem. The application utilizes a highly polished dark-theme aesthetic featuring custom glassmorphism components, responsive multi-layered layouts, dynamic particle backdrops, and seamless animated interactivity.

---

## ✨ Features & Aesthetics

- **Dynamic Hero Section**: Interactive canvas backdrops running lightweight HTML5 particle animations paired with multi-colored ambient aura backlights. Includes custom real-time **Tech Orbit** visualizations wrapping central core frameworks in dual-axis counter-rotating cycles.
- **Glassmorphic UI System**: Tailored card containers rendering deep backdrop blurs, soft multi-layer translucent border lines, and gradient hover states.
- **Icon-Driven Skill Taxonomy**: Streamlined category matrices categorizing modern frontend frameworks, backend web servers, database infrastructure, and DevOps toolchains without clutter.
- **Featured Live Projects Showcase**: Immersive viewports mapping active application deployment screenshots side-by-side with role tags, detailed feature abstracts, stack markers, and dual production/source links.
- **Fully Responsive Architecture**: Fluid container matrices optimized across desktop monitors, tablet displays, and compact mobile screens.

---

## 🛠️ Technology Stack

- **Core Framework**: React 18 powered by Vite fast build utility.
- **Styling System**: Vanilla CSS3 using localized CSS custom properties (variables), native `@keyframes` sequencing, flexbox/grid layout modules, and advanced `object-fit` containment structures.
- **Typography**: Optimized loading paths for Google Fonts **Inter** sans-serif typography suite.
- **Branding Assets**: Direct injection vectors via **Devicons** high-fidelity tech ecosystem CDNs paired with **FontAwesome 4.7** structural interfaces.

## 📦 Getting Started Locally

### Prerequisites
Make sure you have Node.js installed locally on your operating system environment.

### Installation Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ashishyadav9696/simplefolio.git
   cd simplefolio
   ```

2. **Install project dependencies**:
   ```bash
   npm install
   ```

3. **Start the localized local server**:
   ```bash
   npm run dev
   ```

4. **Access Application**:
   Open your default web browser and navigate directly to the output URL provided in your console prompt (typically `http://localhost:5173`).
